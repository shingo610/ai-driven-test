# インバウンド用語 → やさしい言葉 変換辞書（自分用）

## 変換の原則

1. **会話で言える形にする**: 難語を残さず、口頭で自然な言い方にする  
2. **“誰が何をするか”に寄せる**: 行動と場面が浮かぶ表現にする  
3. **初出は解説ボックス**: 図解の前半で必ず解説する  
4. **断定しない**: 国別・属性別は「傾向」「〜と言われる」までに留める  

---

## よく出る変換表

### インバウンド基礎

| 用語・略語 | やさしい言葉 | たとえ（旅行・営業） |
|-----------|-------------|----------------------|
| インバウンド | 日本に来る海外のお客様（訪日客） | 「国内のお客様」の逆 |
| FIT | 個人旅行（オーダーメイド） | 団体バスツアーではなく“自由行動多め” |
| MICE | 会議・報奨・展示会などの団体需要 | 会社イベントの旅行版 |
| ゴールデンルート | 初回訪日で定番の周遊（例: 東京〜富士〜京都〜大阪） | 初めての日本の“王道コース” |
| リピーター | 2回目以降の訪日客 | 「常連さん」 |
| コト消費 | 体験にお金を使う | 物より「体験の思い出」 |
| オーバーツーリズム | 混みすぎ・地域負荷が高い状態 | 繁忙期に受付が回らない状態 |

### デジタル・流通

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| OTA | ネット旅行予約サイト | “ネットの旅行カウンター” |
| DMC | 現地手配のプロ（地域の運営側） | 現地の仕入れ・手配の司令塔 |
| DMO | 地域の観光のまとめ役（戦略・連携） | “地域の観光本部” |
| レビュー | 口コミ評価 | 旅先の「体験の成績表」 |
| 検索流入 | 検索から見つけて来ること | “指名で来店”に近い |

### 需要・季節

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 桜シーズン | 春のピーク需要 | 繁忙期の最上位 |
| 紅葉シーズン | 秋のピーク需要 | 春と並ぶ繁忙期 |
| ショルダー期 | ピークの前後で程よく動く時期 | 混みすぎない“狙い目” |
| ナイトタイム | 夜の観光・消費 | 昼の混雑回避の“別レーン” |
