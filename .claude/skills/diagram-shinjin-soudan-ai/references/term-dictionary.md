# 新人向け：旅行業務（決定〜精算）用語 → やさしい言葉 変換辞書

## 変換の原則

1. **「次に何をするか」が分かる言い方にする**（名詞より動詞）  
2. **不安を減らす**: 「完了条件（何が揃えばOKか）」をセットで書く  
3. **初出は解説ボックス**: 図解本文では必ず用語解説を付ける  
4. **社内ルールは断定しない**: 会社差が出る所は「多くの会社では〜」に留め、確認質問を添える

## よく使う変換表（決定〜精算）

### 見積・条件

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 見積（見積書） | お客様に出す「金額と条件の約束メモ」 | 旅行の行程表の“料金ページ” |
| 条件 | いつまでに何が必要か／変更したらどうなるか | 予約サイトの注意事項 |
| 手数料 | 会社の取り分（作業料） | お店のサービス料 |
| キャンセル料 | 取り消した時に発生するお金 | 予約の“戻らないお金” |
| 変更料 | 日程や人数を変えた時の追加費用 | 切符の変更手数料 |
| 支払条件 | いつ・どうやって払ってもらうか | 支払い期限付きの請求 |

### 手配・在庫

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 手配 | 交通・宿などを「予約して確保する」こと | 席を押さえる |
| 確保 | 在庫を取って“他の人に取られない状態”にする | 椅子に荷物を置いて席取り |
| 在庫 | 予約できる枠（部屋/席/台数） | 空いている座席数 |
| 期日（手配期限） | それまでに予約しないと取れなくなる日 | 締切 |
| 取消 | 予約を取り消すこと | 予約のキャンセル |

### 書類・連絡

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 旅程（行程） | いつ・どこへ・どう移動するかの予定表 | 学校の時間割 |
| 最終案内 | 出発前に渡す“当日の持ち物と集合の決定版” | 当日のしおり |
| バウチャー（Voucher） | 現地で提示する「利用券」 | 入場券/引換券 |
| 確認書 | 内容に間違いがないか確認してもらう紙 | 注文確認メール |

### 請求・入金

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 請求書 | 「この金額をこの期限までに払ってください」の書類 | 支払い用紙 |
| 入金 | お金が振り込まれること | お財布に入る |
| 未収 | まだ入っていないお金（回収待ち） | “まだもらってない代金” |
| 督促 | 入金が遅れているので連絡すること | 支払いリマインド |

### 精算・原価

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 精算 | かかったお金を全部集計して「最後の確定」をする | レシートを全部まとめて家計簿を締める |
| 原価 | 会社が仕入先に払うお金（コスト） | 仕入れ値 |
| 売上 | お客様からもらうお金（売った金額） | 受け取った代金 |
| 粗利 | 売上 − 原価（ざっくり残る分） | 仕入れを引いた手元に残る分 |
| 立替 | 先に会社/担当が払って後で回収する | 一旦建て替える |

## 「聞けない時」に役立つ言い換え（質問に変える）

| そのままだと分からない | 新人が聞ける質問に変換 |
|---|---|
| 条件を詰めて | 「キャンセル料はいつから・いくらですか？変更したら追加は出ますか？」 |
| 手配して | 「いつまでに、どこ（航空/宿/バス）を確保すれば良いですか？」 |
| 精算して | 「原価は何が確定したら締められますか？証拠（請求書/明細）はどれですか？」 |
