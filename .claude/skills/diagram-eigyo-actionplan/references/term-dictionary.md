# 営業・旅行業界用語 → やさしい言葉 変換辞書

## 変換の原則

1. **行動で命名する**: 「誰が何をするか」が伝わる言い方にする  
2. **数字は単位つき**: 前年比・構成比などは一言で補足  
3. **初出は解説ボックス**: SKILL 本線の Phase 2 に合わせて必ず解説  

---

## よく使う変換表

### アクションプラン・管理

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| アクションプラン | 次にやることの一覧と順番 | 旅行の行程表（日付・場所・担当が並ぶ） |
| KPI | 達成度を測る指標 | 体重計の数字（目標に対して今どこか） |
| KGI | 最終的に達成したい成果 | 旅行でいう「旅の目的が達成されたか」 |
| ロードマップ | 時期ごとの予定の全体像 | ガイドマップ上のルート全体 |
| マイルストーン | 大きな区切りの目印 | バスツアーの「昼食地点」「主要観光地」 |
| ボトムアップ | 現場から集めてまとめる | 各係から意見を集めて本社が束ねる |
| トップダウン | 上から方針を渡す | 船長が航路を決めてクルーが実行 |

### 旅行会社・営業現場

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| FIT | 個人旅行の手配 | お客様一人ひとりに合わせたオーダーメイド |
| 団体 | まとまった人数のツアー | 学校行事のバス移動のようなまとまり |
| 仕入れ | 航空券・ホテルなどを会社が確保すること | お店が商品を仕入れるのと同じ |
| 販売単価 | お客様に提示する価格 | メニューに書いてある価格 |
| 粗利 | 売上から原価を引いた利益 | 仕入れ値を引いたあとの手元に残る分 |
| 繁忙期 | 予約が集中する時期 | ゴールデンウィークや年末年始など |
| カウンター | 窓口でお客様と接する場所 | 駅の有人窓口に近い |
| 添乗 | 旅行中同行してサポート | 現地での頼れるガイド役 |

### 社内・連携

| 用語・略語 | やさしい言葉 | たとえ |
|-----------|-------------|--------|
| 営業支援 | 営業の事務・資料・連絡を助ける役 | 舞台裏で道具を整えるスタッフ |
| エスカレーション | 判断を上に上げること | 店長に相談を持ち上げる |
| 横串 | 部門をまたいだ取りまとめ | 複数の窓口をつなぐ総合デスク |
